#Manufacturing Defect Detection
Machine Learning project for predicting manufacturing defects using production process data. 
The goal of this project is to simulate an industrial workflow from data generation to model deployment while following software engineering best practices. 

1. Overview
Predict manufacturing defects using machine learning models trained on production process data.

2. Project Goals
    Generate synthetic manufacturing data
    Build an ML classification model
    Evaluate model performance
    Visualize results
    Deploy the model (optional)

3. Tech Stack

    NumPy

    GitHub
